Beat Saber Troubleshooting
Having trouble with patching Beat Saber? Don't worry you are not alone. The patching process is quite involved and things can go wrong. Thankfully there are things you can do!

Scenario 1
Do you see any of these errors?
System.ApplicationException: Either the 'levels' or...
System.IO.FileNotFoundException: Could not find file...

Select this option to reset your song packs file. Also be advised that any special characters in the file path can cause the patcher to fail. If your user folder path has any éíúóá or other special characters in it you may need to use another user account on your PC for SideQuest.
Reset Packs


Scenario 2
Do you see any of these errors?
Error: Command failed:...
System.ApplicationException: Could not find...
LibSaberPatch.SerializedAssets+ParseException:...
Ionic.Zip.ZipException: Cannot read that as a...

You can fix any of these errors with the following steps. They are ordered from easiest to hardest. Try to sync songs between each solution.
- Solution 1
Reset your working draft APK to the clean master backup.
Reset Patched To Master


- Solution 2
Backup a new master copy ( this requires a clean copy of beat saber on your headset. If you are not sure skip to - Solution 3. )

First:
Backup New Master

Then:
Reset Patched To Master


- Solution 3
This is similar to the above but includes some extra steps.

First:
Uninstall Beat Saber ( using the apps icon above apps ).

Then:
Put on your headset and reinstall it from the Oculus Store

Then:
Backup New Master

Then:
Reset Patched To Master
